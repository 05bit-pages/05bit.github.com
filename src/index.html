Интро

Мы разрабатываем веб-сервисы, в основе которых есть новая оригинальная идея.

Любим технологии и используем лучшие из них — те, с которыми приятно работать, которые эффективны, которые созданы передовыми инженерами мирового уровня:

  • Python / Django / Flask
  • Javascript / jQuery / KnockoutJS / Bootstrap
  • "Облачные" сервисы Amazon и Google

Мы максимально используем возможности существующих сервисов, не изобретая велосипеды. Знаем кун-фу платежных систем и авторизации через социальные сети.
