Welcome!
========

This is [05bit.com](http://05bit.com) repo for hosting website at GitHub Pages.

Install and develop
-------------------

1. Clone this repo
2. Install Python dependencies

    ```
    pip install docta
    ```
3. Run dev server

    ```
    docta serve --watch
    ```
4. Add or edit files, auto-rebuild should do the rest
5. Deploy with `git push`
